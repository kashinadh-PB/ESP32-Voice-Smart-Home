# ESP32 Voice Controlled Smart Home

This project controls multiple appliances using ESP32, Google Assistant and SinricPro.

Devices:
- Light
- Fan
- Switch

Hardware:
- ESP32
- 3 LEDs
- 5161AS 7-segment display
- Resistors

Features:
- Voice control using Google Home
- IoT control via SinricPro
- 7-segment display shows device number
